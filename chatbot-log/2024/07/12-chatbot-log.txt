      ```
      ├── app.py (メインのStreamlitアプリファイル)
      ├── requirements.txt (必要なPythonパッケージ)
      ├── setup.sh (Streamlit起動スクリプト)
      └── Procfile (Herokuに指示を出すファイル)
      ```
      
2. **`requirements.txt` の作成**
    - `requirements.txt` ファイルには必要なパッケージをリストアップします。例えば:

      ```sh
      mkdir -p ~/.streamlit/

      echo "\
      [server]\n\
      headless = true\n\
      port = $PORT\n\
      enableCORS = false\n\
      \n\
      " > ~/.streamlit/config.toml
      ```

4. **`Procfile` の作成**
    - `Procfile` には以下のように記述します。これはHerokuがどのコマンドを実行するかを指示するファイルです。

      ```sh
      git init

      git commit -m "Initial commit"
      ```

6. **Herokuで新しいアプリを作成**
    - 以下のコマンドを実行してHeroku上で新しいアプリを作成します。
      ```sh
      heroku create
      ```

7. **アプリをデプロイ**
    - 以下のコマンドでHerokuにプッシュし、アプリをデプロイします。
      ```sh
      git push heroku master
      ```

8. **デプロイの確認**
    - デプロイが成功すると、HerokuからURLが提供されます。そのURLにアクセスしてアプリを確認します。

1. **Streamlitアプリの準備**：
   まず、Streamlitアプリが動作することを確認します。
   ```bash
   streamlit run your_app.py
   ```
   ここで、`your_app.py` はあなたのStreamlitアプリのファイル名です。

2. **XAMPPのインストールと起動**：
   XAMPPをインストールし、Apacheサーバーを起動します。XAMPPコントロールパネルで「Start」ボタンをクリックします。

3. **StreamlitアプリをCGIスクリプトとして設定**：
   ApacheがCGIスクリプトを実行できるように設定します。

   - `httpd.conf` ファイルを編集（通常は `C:\xampp\apache\conf\httpd.conf`）：
     ```apache
     # Add these lines to enable CGI scripts
     ScriptAlias /cgi-bin/ "C:/xampp/cgi-bin/"
     <Directory "C:/xampp/cgi-bin">
         AllowOverride None
         Options +ExecCGI
         Require all granted
         AddHandler cgi-script .py
     </Directory>
     ```

4. **Pythonスクリプトの作成**：
   `C:\xampp\cgi-bin\` ディレクトリに移動し、ここにStreamlitアプリを起動するためのスクリプトを作成します。
   例：`run_streamlit_app.py`
   ```python
   #!/usr/bin/env python
   import os
   os.system("streamlit run C:/path/to/your_app.py --server.port 8501 --server.enableCORS false")
   ```

   必要に応じてパス（C:/path/to/your_app.py）を修正してください。

5. **スクリプトに実行権限を付与**：
   スクリプトに実行権限を付与します。Windowsでは通常不要ですが、念のため確認してください。

6. **Apacheを再起動**：
   設定を反映させるためにApacheを再起動します（XAMPPコントロールパネルで「Stop」ボタンを押し、その後「Start」ボタンを押します）。

7. **アプリケーションのアクセス**：
   ブラウザで以下のURLにアクセスし、Streamlitアプリの動作を確認します。
   ```
   http://localhost/cgi-bin/run_streamlit_app.py
   ```
